# Rust迷你内核项目说明

## 项目简介

本目录包含一个使用Rust语言实现的简易版类Linux内核项目。这是一个教育性质的项目，旨在展示如何使用Rust进行操作系统内核开发。

## 项目位置

- 目录: `/rust_mini_kernel`
- 语言: Rust
- 类型: 裸机操作系统内核

## 主要特性

1. **无标准库环境** (`no_std`): 完全不依赖操作系统，直接在裸机上运行
2. **VGA文本输出**: 直接操作显卡内存进行文本显示
3. **串口通信**: 用于调试和日志输出
4. **自定义panic处理**: 内核级别的错误处理机制
5. **中英文双语支持**: 启动界面显示中英文信息

## 技术亮点

### 为什么选择Rust？

- **内存安全**: Rust的所有权系统在编译期保证内存安全，避免了传统C/C++内核中常见的内存错误
- **零成本抽象**: 提供高级语言特性的同时保持与C相当的性能
- **现代化工具链**: Cargo提供了优秀的包管理和构建体验
- **强类型系统**: 编译期捕获大量潜在错误

### 核心概念

1. **裸机编程**: 不依赖任何操作系统API，直接与硬件交互
2. **VGA缓冲区**: 通过内存映射I/O访问显卡内存（地址0xb8000）
3. **自旋锁**: 使用无锁数据结构保证并发安全
4. **链接器脚本**: 精确控制内核代码在内存中的布局

## 快速开始

```bash
# 进入项目目录
cd rust_mini_kernel

# 安装依赖（首次运行）
make install-deps

# 构建内核
make build

# 在QEMU中运行
make run
```

详细文档请查看 `rust_mini_kernel/README.md`
