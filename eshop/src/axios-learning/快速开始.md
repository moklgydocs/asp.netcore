## 📝 快速测试封装的工具

### 在任意 Vue 组件中：

```vue
<script setup>
import request from '@/utils/request'

// GET 请求
const fetchData = async () => {
  const data = await request.get('/api/posts')
  console.log(data)
}

// POST 请求
const createPost = async () => {
  const result = await request.post('/api/posts', {
    title: 'Hello',
    content: 'World'
  })
  console.log(result)
}

// 文件上传
const uploadFile = async (file) => {
  const result = await request.upload(
    '/api/upload',
    file,
    (percent) => console.log(`进度：${percent}%`)
  )
  console.log(result)
}
</script>
```

## ✨ 特色功能

### 1. 自动 Token 管理
```typescript
// 自动添加 Token，自动刷新，无需手动处理
const data = await request.get('/api/user/info')
```

### 2. 智能错误处理
```typescript
// 自动显示错误提示，支持重试
const data = await request.get('/api/posts', {
  retryCount: 3,      // 失败重试3次
  showError: true     // 显示错误提示
})
```

### 3. 请求取消
```typescript
// 自动取消重复请求，无需手动管理
request.get('/api/search?q=vue')
request.get('/api/search?q=react')  // 自动取消上一个
```

### 4. Loading 管理
```typescript
// 自动显示和隐藏 Loading
const data = await request.get('/api/posts', {
  showLoading: true
})
```

### 5. 类型安全
```typescript
// 完整的 TypeScript 类型支持
const posts = await request.get<Post[]>('/api/posts')
```
